## node-static server

http static server (nodejs)

```
npm install -g node-static
static
```

python

```
python -m SimpleHTTPServer
```

atom editor

```
atom -> Packages -> LiveReload -> Toggle Enable
```

browser

```
browser to http://localhost:8080/
chrome -> enable livereload
```
